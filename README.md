# Предисловие
ASalary - Плагин, выдающий награду в виде монет или предметов за наигранное время.

**Версия: 1.16.5-1.21.4 (Paper)**

> [!IMPORTANT]
> Плагин требует PlaceholderAPI, Vault и экономический плагин (пр. EssentialsX)

# Функционал
Плагин имеет гибкую настройку, добавление зарплат за разное кол-во наигранного времени.
Достаточно лишь выдать указанный в конфиге пермишн игроку или группе.
После создания зарплаты и выдачи пермишна нужно перезагрузить плагин и перезайти на сервер.

# Конфиг
Базовый конфиг представлен ниже
```
Messages:
  reload: "&aПлагин успешно перезагружен"
Salaries:
  default:
    time: 60
    permission: "ASalary.default"
    money: 2000
    message:
      - "&aВы получили &e2000 &aмонет и алмаз"
      - "&aза 1 минуту на сервере"
      - "&aТеперь ваш баланс: %vault_eco_balance%"
    commands:
      - "give {player} diamond 1"
  vip:
    time: 60
    permission: "ASalary.vip"
    money: 10000
    message:
      - "&#FFF000Вы получили 10000"
      - "&#FFF000за 1 минуту на сервере"
      - "&#FFF000Теперь ваш баланс: %vault_eco_balance%"
    commands: []
```

# Возможности
В конфиге можно настроить сообщения при выдаче зарплаты, обычные и HEX цвета, команды которые выполнятся,
а так же то, сколько нужно наиграть времени (в секундах), кол-во монет при получении зп.
Вы можете создавать сколько угодно видов зарплат, главное указывать уникальный пермишн и не забывать его выдать на сервере.

# Плейсхолдеры
В плагине поддерживаются плейсхолдеры.

# Кратко о командах и разрешениях

> ### Команды
**/asalary reload** - Перезагрузить плагин.

> ### Разрешения
**asalary.*** - Доступ ко всем командам.\
**asalary.reload** - Доступ к /asalary reload.\
